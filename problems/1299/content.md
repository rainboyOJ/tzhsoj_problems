## 描述

金猪大家庭主要由母猪构成的，小母猪第二年就可以生2只小母猪和2只小公猪，第三年开始每年可以生4只小母猪和4只小公猪；小公猪在第二年就要被赶出大家庭；猪的寿命只有六年。 目前，一个新的金猪大家庭正在形成：一只小母猪自立门户，创建自己的金猪大家庭。 

## 输入格式

只有一行且只有一个正整数：n (1<=n<=35) 。表示n年后。

## 输出格式

输出只有一行且只有一个正整数m。（表示n年后金猪大家庭所有成员的数量）

## 输入样例

```plaintext
2
```

## 输出样例

```plaintext
5
```
