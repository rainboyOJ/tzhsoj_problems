## 描述

给定平面上N个点，你需要计算以其中4个点位定点的正方形的个数。 注意这里的正方形边不一定需要和坐标轴平行。

## 输入格式

第一行一个整数：N； 接下来，共N行，每行两个整数，空格隔开，用来表示N个点的坐标。 

## 输出格式

一个数，用来表示正方形的个数。

## 输入样例

```plaintext
7 0 0 0 1 1 0 1 1 1 2 2 1 2 2 
```

## 输出样例

```plaintext
3 
```

Hint

对于20%的数据，满足1≤N≤20； 对于100%的数据，满足1≤N≤500，-50≤X[i]，Y[i]≤50，点不会重叠。
