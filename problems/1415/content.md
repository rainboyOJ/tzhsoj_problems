## 输入格式

输入数据可能存在多个。 对于每一个数据n而言，均有（2<=n<=10000）。

## 输出格式

对于其中每一个数据n： 输出从2到n之间所有的亲密数对（a，b），且a小于b。 每行仅输出一对亲密数对（对于一个数据n，可能有多行输出）。 具体输出格式请参看样例。

## 输入样例

```plaintext
500 1000
```

## 输出样例

```plaintext
n:500 (220,284) n:1000 (220,284) 
```
