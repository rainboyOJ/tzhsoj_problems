## 输入格式

输入一行字符串，并以字符“;”结束。

## 输出格式

对不同的输入数据对应以下三种处理结果： （1）右括号先出现输出：error 1:right brackets is first appear! （2）左括号比右括号多：error 2:left brackets are more then right brackets! （3）左括号与右括号匹配成功：OK!

## 输入样例

```plaintext
((a+b)*c/d)/2;
```

## 输出样例

```plaintext
OK! 
```
