## 描述

甲、乙两人同时从A地出发要尽快同时赶到B地。出发时A地有一辆小车，可是这辆小车除了驾驶员外只能带一人。已知甲、乙两人的步行速度一样，且小于车的速度。问：怎样利用小车才能使两人尽快同时到达。

## 输入格式

仅一行，三个数据分别表示AB两地的距离s，人的步行速度a，车的速度b。

## 输出格式

两人同时到达B地需要的最短时间。

## 输入样例

```plaintext
car.in 120 5 25 
```

## 输出样例

```plaintext
car.out（格式：输出时要求小数点保留两位输出） 9.60 
```
