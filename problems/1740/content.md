## 描述

小明在你的帮助下，破密了Ferrari设的密码门，正要往前走，突然又出现了一个密码门，门上有一个算式，其中只有“(”，“)”，“0-9”，“+”，“-”，“*”，“/”，“^”，求出的值就是密码。小明数学学得不好，还需你帮他的忙。(“/”用整数除法)

## 输入格式

输入文件calc.in共1行，为一个算式。

## 输出格式

输出文件calc.out共1行，就是密码。

## 输入样例

```plaintext
1+(3+2)*(7^2+6*9)/(2)
```

## 输出样例

```plaintext
258
```

Hint

100%的数据满足：算式长度<=30 其中所有数据在2<sup>31</sup>-1的范围内。
