## 输入格式

输入数据可能存在多个。

## 输出格式

分多行输出。 每行仅输出，对应x的值及其对应的s值。（每对数据中间使用空格隔开，小数点后均保留五位） 具体输出格式请参看样例。

## 输入样例

```plaintext
0.874 -0.992 -0.232 0.1234
```

## 输出样例

```plaintext
x=0.87400 s=1.39648 x=-0.99200 s=-0.62917 x=-0.23200 s=-0.20705 x=0.12340 s=0.13134 
```
